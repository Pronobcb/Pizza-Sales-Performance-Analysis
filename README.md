# 🍕 Pizza Sales Performance Analysis

## 📌 Project Overview
This project presents an interactive **Pizza Sales Dashboard** designed to analyze sales performance and business trends. The dashboard provides clear insights into revenue, order patterns, and product performance, helping to understand key factors driving sales.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top and bottom performing pizzas  
- Understand sales trends over time (daily & monthly)  
- Evaluate the impact of category and size on sales  
- Support data-driven business decisions  

---

## 📊 Key Metrics
- **Total Revenue:** $817.86K  
- **Total Orders:** 21,350  
- **Total Pizzas Sold:** 49,574  
- **Average Order Value:** 38.31  
- **Average Pizza per Order:** 2.32

## 🔍 Key Insights
- Sales are highest on **weekends**, especially **Friday**  
- **July and May** show peak monthly performance  
- **Classic category** contributes the most to total sales  
- **Large size pizzas** generate the highest revenue share  
- Some pizzas consistently perform as **top sellers**, while others show low demand
